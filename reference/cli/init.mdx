## Description

Initialize a new Forest Admin project or connect an existing application.

## Sources to migrate

- CLI init documentation
- Getting started guides

## Content outline

1. **Command Overview**
   ```bash
   forest init
   ```

2. **What It Does**
   - Creates Forest Admin configuration
   - Generates `.forestadmin-schema.json`
   - Sets up environment variables
   - Connects to your project

3. **Interactive Prompts**
   - Project selection
   - Environment selection
   - Database connection
   - Framework detection

4. **Prerequisites**
   - Forest Admin account
   - Logged in (`forest login`)
   - Database configured
   - Application directory

5. **Output Files**
   - `.forestadmin-schema.json`
   - `.env` updates (environment variables)
   - Configuration files

6. **Options**
   - `--project <id>` - Specify project
   - `--environment <name>` - Specify environment
   - `--skip-db` - Skip database configuration

7. **After Init**
   - Start your application
   - View in Forest Admin UI
   - Next steps

8. **Examples**
   ```bash
   # Interactive init
   forest init

   # With specific project
   forest init --project abc123

   # Skip database setup
   forest init --skip-db
   ```

9. **Troubleshooting**
   - Not logged in
   - Database connection errors
   - Permission issues
   - Project not found
