**Package:** `forest-express-sequelize` v9

**Migration Guide:** [See Migration Guide](/guides/migration/from-v1/overview)

## Why Migrate?

### Agent v2 Improvements

**Better Performance:**
- Faster query execution
- Optimized data loading
- Reduced memory footprint

**New Features:**
- Native TypeScript support (Node.js)
- Better plugin system
- Improved customization API
- Cross-data source relationships
- Better error handling

**Better Developer Experience:**
- Cleaner API
- Better documentation
- Active development and support
- Regular updates and new features

**Security:**
- Active security patches
- Modern authentication
- Better permission system

### Migration Support

We provide:
- ✅ Detailed migration guides
- ✅ Migration tools and scripts
- ✅ Support during migration
- ✅ Side-by-side running (v1 + v2 in parallel)

**Start migration:** [Migration Guide](/guides/migration/from-v1/overview)
