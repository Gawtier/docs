Forest Admin is built on 4 foundational layers that work together to help you build powerful operational tools for your teams.

## Operations

Give teams the tools to run day-to-day ops efficiently.

### Build

Create interfaces tailored to each team's workflow.

| Feature | Description |
|---------|-------------|
| [Workspaces](/product/build/workspaces) | Create dedicated interfaces for different teams (Support, Sales, Operations) |
| [Collections & Fields Layout](/product/build/layout-editor) | Customize collections, tables, forms, and field layouts without code |
| [Summary Views](/product/build/custom-views/summary-views) | Display aggregated data and key metrics at the top of collections |
| Analytics per Record | Show charts and analytics for individual records |
| Explorer View | Navigate and explore relationships between records visually |
| [Smart Views](/product/build/custom-views/smart-views) | Build custom components for advanced use cases |

### Execute

Run operations and take actions on your data.

| Feature | Description |
|---------|-------------|
| [Smart Actions](/product/process/actions/overview) | Custom buttons that trigger server-side logic |
| [Browse & Search](/product/execute/browse) | Advanced filters, search, and data exploration |
| Workflows | Multi-step automated processes |

### Collaborate

Work together as a team with built-in collaboration tools.

| Feature | Description |
|---------|-------------|
| [Inbox](/product/collaborate/inbox) | Assign tasks and track work across your team |
| [Approval Workflows](/product/collaborate/approval-workflows) | Route actions through approval chains |
| [Notes & Comments](/product/collaborate/notes) | Discuss and document decisions inline |

### Manage

Monitor performance and track what's happening.

| Feature | Description |
|---------|-------------|
| [Dashboards](/product/manage/dashboards) | Custom analytics and KPI tracking |
| Smart Charts | Create custom charts with aggregations and time series |
| [Activity Feed](/product/manage/activity) | Real-time view of team actions |

### Embed

Integrate Forest Admin into your existing tools.

| Feature | Description |
|---------|-------------|
| Forest Assist | AI assistant for your operations |

## Process

Codify business logic so ops run consistently.

| Feature | Description | For |
|---------|-------------|-----|
| No-Code Actions | Create actions through visual interface | Ops Teams |
| MCP Connections | Connect to external tools via Model Context Protocol | Ops Teams |
| Process Builder | Design multi-step workflows visually | Ops Teams |
| [Smart Actions](/product/process/actions/overview) | Execute custom server-side logic | Tech Teams |
| [Smart Fields](/product/process/fields/overview) | Add computed fields to your collections | Tech Teams |
| [Smart Segments](/product/process/segments/overview) | Create pre-filtered views of your data | Tech Teams |

## Control

Keep ops secure and compliant at scale.

| Feature | Description | Category |
|---------|-------------|----------|
| [Roles & Permissions](/product/control/roles-permissions) | Define granular access controls per collection and action | Access Control |
| [Scopes](/product/control/scopes-and-visibility) | Filter what users see based on their role | Access Control |
| [SSO & SAML](/product/control/authentication/sso) | Integrate with Google, Okta, Azure AD, and more | Authentication |
| [Two-Factor Auth](/product/control/authentication/2fa) | Add extra security with 2FA | Authentication |
| [SCIM Provisioning](/product/control/authentication/scim) | Auto-provision users from your IdP | Authentication |
| [IP Whitelisting](/product/control/security/ip-whitelisting) | Restrict access to specific IP addresses | Security |
| [Audit Logs](/product/control/audit) | Track who did what, when, and from where | Audit |

## Integration

Connect any data, wherever it lives.

| Feature | Description | Type |
|---------|-------------|------|
| [SQL Databases](/product/integration/data-sources/sql) | PostgreSQL, MySQL, SQL Server, SQLite | Data Source |
| [NoSQL Databases](/product/integration/data-sources/nosql) | MongoDB and other document stores | Data Source |
| [REST APIs](/product/integration/data-sources/rest-api) | Connect to any REST API as a data source | Data Source |
| [Custom Sources](/product/integration/data-sources/custom) | Build adapters for any data source | Data Source |
| Multiple Sources | Connect multiple databases in one admin panel | Data Source |
| [Architectures](/product/integration/architectures/self-hosted) | Cloud, Self-Hosted, or On-Premise deployment | Deployment |
