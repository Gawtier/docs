## Description

Choose your framework and follow the setup guide to connect Forest Admin to your application.

## Framework Priority

We recommend the following priority order:

1. **[Node.js](/product/integration/setup/nodejs)** ← PRIORITY 1 (Most features, best support)
2. **[Ruby](/product/integration/setup/ruby)** ← PRIORITY 2 (Mature, Rails integration)
3. **[Python](/product/integration/setup/python)** ← PRIORITY 3 (Django/Flask support)

## Quick Setup Summary

### Node.js (@forestadmin/agent)
```bash
npm install @forestadmin/agent
```
- TypeScript support
- Express/Fastify/NestJS integration
- Most comprehensive features

### Ruby (forest_liana)
```bash
gem install forest_liana
```
- Rails integration
- ActiveRecord support
- Mature and stable

### Python (forestadmin-agent-python)
```bash
pip install forestadmin-agent-python
```
- Django/Flask support
- SQLAlchemy/Django ORM
- Growing feature set

## What You'll Learn

Each setup guide covers:
