## Overview

The Security tab provides essential security features for project management.

<Info>
Some features may not be available on all plans. Check the [pricing page](https://www.forestadmin.com/pricing) for more information, or [contact us](https://www.forestadmin.com/contact) to enable them.
</Info>

## Available Security Features

The Security tab provides four key security features:

<CardGroup cols={2}>
  <Card title="IP Whitelisting" icon="network-wired" href="/product/control/security/ip-whitelisting">
    Control which IP addresses can access your admin panel
  </Card>

  <Card title="Auto Logout" icon="clock" href="/product/control/security/auto-logout">
    Force user session logout after a few minutes of inactivity
  </Card>

  <Card title="Two-Factor Authentication (2FA)" icon="shield-halved" href="/product/control/security/2fa">
    Enforce a second authentication factor for specific environments
  </Card>

  <Card title="User Provisioning (SCIM)" icon="users" href="/product/control/security/scim">
    Scale user management through Identity Providers
  </Card>
</CardGroup>
